# microkv-cli

Defines a CLI application that runs as a portable key-value store service
on top of any volume.


## build:

With Docker:

```
$ docker build -t .
```

Install locally:

```
$ cargo install --path .
```

At this point, the `microkv-cli` is ready for use either as a server or a client CLI.

## usage

```
$ micro-kv --help
```
